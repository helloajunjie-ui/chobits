## 💡 安装建议

如果需要安装缺失的工具，推荐方式：

1. **ffmpeg** → 访问 [ffmpeg.org](https://ffmpeg.org/) 下载，或使用 `winget install ffmpeg`
2. **nmap** → 访问 [nmap.org](https://nmap.org/) 下载，或使用 `winget install nmap`
3. **yt-dlp** → `pip install yt-dlp`（如果未安装）
4. **gcc** → 安装 MinGW 或 Visual Studio Build Tools
